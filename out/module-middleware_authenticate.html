Methods

(inner) checkAuthentication(req, res, next)

Middleware function to check user authentication. This function checks whether the `req.session.user` exists. If it does, the request proceeds to the next middleware or route handler. If not, the user is redirected to the login page.
Parameters:
Name Type Description
req Object Express request object. Contains the session data.
res Object Express response object. Used to redirect unauthenticated users.
next function Function to pass control to the next middleware or route handler if authenticated.
