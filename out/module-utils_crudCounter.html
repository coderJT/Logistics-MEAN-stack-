Module: utils/crudCounter

This module provides functions to increment the counters for Create, Read, Update, and Delete (CRUD) operations in Firestore. Each function targets the `crud/operationCount` document and increments the respective field by 1 using Firestore's increment operation. - The counters are used to track how many times each type of operation (Create, Read, Update, Delete) is performed in the application.
